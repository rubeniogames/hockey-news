# RoofCalc Telegram Web App (MVP)

Минимальная версия веб-приложения для Telegram Mini App:

- расчет двускатной крыши для частного дома;
- учет снеговой и ветровой нагрузок (вручную, по данным региона);
- автоподбор сечения стропил из фиксированного набора;
- проверка по напряжению и прогибу;
- 2D-схема крыши;
- ориентировочный объем древесины по стропилам.

## Что это сейчас

Это **первый MVP**, чтобы быстро протестировать процесс с реальными входными данными.
Расчет упрощен и не заменяет полноценный конструкторский проект.

## Как запустить локально

```bash
python3 -m http.server 8080
```

Открыть:

- `http://localhost:8080`
